Problem Link: https://leetcode.com/problems/reverse-integer

Solution in Java:

```
class Solution {
    public int reverse(int x) {

        boolean isNegative = x < 0;

        if (isNegative) {
            x *= -1;
        }

        String num = "" + x;

        String answer = "";
        for (int i = 0; i < num.length(); i++) {
            answer += num.charAt(num.length() - i - 1);
        }

        if (isNegative) {
            answer = "-" + answer;
        }

        try {
            return Integer.parseInt(answer);
        } catch (Exception ex) {
            return 0;
        }

    }
}
```
